# Find duplicates in a customer table

### Install

```sh
$ npm install
$ mvn clean install
```

### Run application during dev

#### run server
```sh
$ java -jar target/react-starter-0.1.0.jar
```

or by running Application class from your IDE.

#### run hot reload server
```sh
$ npm start
```
